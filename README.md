# Obtener el espacio  más grande de Ceros en un número binario.

El algoritmo a desarrollar debe obtener la distancia más grande de ceros entre un par
de unos de un número binario.

Ejemplos:

| Binario  | Gap  | Distancia   |
|---|---|---|---|---|
|10001001|10001   | 3  |
|1000|   | 0  |
|1111|   | 0  |
